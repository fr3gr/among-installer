# Among Us Mod Installer

Automatyczny instalator modów do Among Us.

## Pobieranie

[**Download najnowszej wersji**](https://github.com/fr3gr/among-installer/releases/latest)

## Jak używać

1. Rozpakuj ZIP
2. Uruchom `AmongUsModInstaller.exe`
3. Kliknij **Automatyczna Instalacja**
4. Gotowe!

## Funkcje

- Automatyczne wykrywanie gry (Steam/Epic/MS Store)
- Auto-update aplikacji
- System backupów
- Restore i Uninstall

## Wymagania

- Windows 10/11
- .NET 9.0 Runtime (pobierze się automatycznie)
